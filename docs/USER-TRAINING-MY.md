# NexoraSIM Enterprise အသုံးပြုသူ လေ့ကျင့်ရေး လမ်းညွှန်

## စတင်ခြင်း

### Portal ဝင်ရောက်ခြင်း
1. ဝက်ဘ်ဆိုဒ်သို့ သွားပါ: https://nexorasim.powerappsportals.com
2. ကုမ္ပဏီ Microsoft အကောင့်ဖြင့် ဝင်ရောက်ပါ
3. MFA အတည်ပြုခြင်းကို ပြီးမြောက်စေပါ

### စက်ပစ္စည်း တပ်ဆင်ခြင်း

#### Windows 10/11
1. Portal တွင် **စက်ပစ္စည်း စတင်မည်** ကို နှိပ်ပါ
2. **Windows** platform ကို ရွေးပါ
3. သင့်ဆက်သွယ်ရေးကုမ္ပဏီကို ရွေးပါ (MPT/ATOM/U9/MYTEL)
4. VPN profile သည် Intune မှတစ်ဆင့် အလိုအလျောက် တပ်ဆင်မည်
5. Always On VPN အလိုအလျောက် စတင်မည်

#### iOS/Android
1. Microsoft Company Portal app ကို ထည့်သွင်းပါ
2. ကုမ္ပဏီ အထောက်အထားများဖြင့် ဝင်ရောက်ပါ
3. VPN profile အလိုအလျောက် တပ်ဆင်မည်
4. VPN ဖွဲ့စည်းမှုကို လက်ခံပါ

### eSIM စီမံခန့်ခွဲမှု

#### eSIM Profile ဒေါင်းလုဒ်လုပ်ခြင်း
1. **eSIM စီမံခန့်ခွဲမှု** အပိုင်းသို့ သွားပါ
2. ဆက်သွယ်ရေးကုမ္ပဏီကို ရွေးပြီး ICCID ကို ရိုက်ထည့်ပါ
3. **Profile ဒေါင်းလုဒ်လုပ်မည်** ကို နှိပ်ပါ
4. QR code ကို စကင်န်ဖတ်ပါ သို့မဟုတ် activation code ကို အသုံးပြုပါ

#### eSIM အသုံးပြုခြင်း
1. ICCID နှင့် activation code ကို ရိုက်ထည့်ပါ
2. ဆက်သွယ်ရေးကုမ္ပဏီကို ရွေးပါ
3. **အသုံးပြုမည်** ကို နှိပ်ပါ
4. Profile သည် ၂ မိနစ်အတွင်း အသုံးပြုနိုင်မည်

### Dashboard အသုံးပြုခြင်း

#### ခွဲခြမ်းစိတ်ဖြာမှု ကြည့်ရှုခြင်း
- **အသုံးပြုမှု စာရင်းအင်း**: ဆက်သွယ်ရေးကုမ္ပဏီအလိုက် ဒေတာ သုံးစွဲမှု
- **ချိတ်ဆက်မှု အခြေအနေ**: VPN နှင့် eSIM အခြေအနေ
- **လိုက်နာမှု အစီရင်ခံစာများ**: လုံခြုံရေး လိုက်နာမှု အခြေအနေ

#### ဘာသာစကား ပြောင်းလဲခြင်း
- ဘာသာစကား dropdown (ညာဘက်အပေါ်) ကို နှိပ်ပါ
- English သို့မဟုတ် မြန်မာ ကို ရွေးပါ
- Interface အလိုအလျောက် အပ်ဒိတ်ဖြစ်မည်

### ပြဿနာ ဖြေရှင်းခြင်း

#### VPN ပြဿနာများ
- Portal တွင် စက်ပစ္စည်း လိုက်နာမှုကို စစ်ဆေးပါ
- လိုအပ်ပါက VPN service ကို ပြန်လည်စတင်ပါ
- အခြေအနေက "လိုက်နာမှုမရှိ" ပြပါက IT ကို ဆက်သွယ်ပါ

#### eSIM ပြဿနာများ
- ICCID ပုံစံကို စစ်ဆေးပါ (ဂဏန်း ၂၀ လုံး)
- ဆက်သွယ်ရေးကုမ္ပဏီ ရရှိနိုင်မှုကို စစ်ဆေးပါ
- ၅ မိနစ်အကြာတွင် ပြန်လည်ကြိုးစားပါ
