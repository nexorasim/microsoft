# NexoraSIM Enterprise User Training Guide

## Getting Started

### Portal Access
1. Navigate to: https://nexorasim.powerappsportals.com
2. Sign in with your company Microsoft account
3. Complete MFA authentication when prompted

### Device Setup

#### Windows 10/11
1. Click **Initialize Device** in the portal
2. Select **Windows** platform
3. Choose your carrier (MPT/ATOM/U9/MYTEL)
4. VPN profile will auto-deploy via Intune
5. Always On VPN activates automatically

#### iOS/Android
1. Install Microsoft Company Portal app
2. Sign in with company credentials
3. VPN profile deploys automatically
4. Accept VPN configuration when prompted

### eSIM Management

#### Download eSIM Profile
1. Go to **eSIM Management** section
2. Select carrier and enter ICCID
3. Click **Download Profile**
4. Scan QR code or use activation code

#### Activate eSIM
1. Enter ICCID and activation code
2. Select carrier
3. Click **Activate**
4. Profile activates within 2 minutes

### Dashboard Usage

#### View Analytics
- **Usage Statistics**: Data consumption by carrier
- **Connection Status**: VPN and eSIM status
- **Compliance Reports**: Security compliance status

#### Language Switch
- Click language dropdown (top right)
- Select English or မြန်မာ
- Interface updates automatically

### Troubleshooting

#### VPN Issues
- Check device compliance in portal
- Restart VPN service if needed
- Contact IT if status shows "Non-Compliant"

#### eSIM Issues
- Verify ICCID format (20 digits)
- Check carrier availability
- Retry activation after 5 minutes
