# NexoraSIM eSIM Enterprise Management

Secure enterprise-grade eSIM lifecycle management platform with iOS and Android direct installation capabilities.

## Features

### eSIM Management
- Complete eSIM information display (ICCID, Network, Status, Data Plan)
- QR code generation for device installation
- Phone number assignment and transfer
- Direct iOS and Android eSIM installation

### Device Support
- iOS 12.1+ with Apple Business Manager integration
- Android 9+ with enterprise MDM support
- Direct eSIM installation buttons for both platforms
- Cross-platform compatibility

### Enterprise Features
- Microsoft Entra ID authentication
- Role-based access control
- Bulk provisioning capabilities
- Real-time analytics dashboard
- Comprehensive audit logging

### Security
- 100% secure deployment
- Encrypted data transmission
- Immutable audit trails
- Compliance with enterprise standards

## Installation Process

### iOS Devices
1. Generate QR code or use direct install button
2. Automatic redirect to iOS eSIM setup
3. Profile installation via Apple Business Manager
4. Immediate activation and phone number assignment

### Android Devices
1. Direct installation through enterprise MDM
2. Automatic profile configuration
3. Network activation and number transfer
4. Enterprise policy enforcement
